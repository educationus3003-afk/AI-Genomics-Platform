2. Analysis basis and workflow

  • Small-variant processing: bcftools normalization → VEP 115 → SnpEff → ClinVar annotation → SpliceAI.
  • CNV processing: AnnotSV → ClassifyCNV → ISV-CNV with SHAP values → combined CNV summary.
  • Sequence-variant terminology follows the ACMG/AMP five-tier framework [5].
  • CNV classification is interpreted in the context of the ACMG/ClinGen evidence-based framework [6].
  • The report does not independently recalculate ACMG criteria; it summarizes classifications generated by the supplied files.

Integrated interpretation

The KRT86 SNV is the strongest finding because it is a heterozygous missense variant annotated as pathogenic in ClinVar and fits the dominant keratin-gene mechanism. The DSG4-containing deletion is biologically relevant, but the CNV tools disagree: AnnotSV classified it as pathogenic, ClassifyCNV as uncertain significance, and ISV remained below the selected 0.95 threshold. In a real case, zygosity and a second DSG4 pathogenic allele would be essential before attributing an autosomal recessive DSG4-related diagnosis to this deletion.

Educational follow-up for a real clinical specimen

For a real specimen: confirm the KRT86 variant, correlate with hair-shaft microscopy/dermatologic findings, and perform segregation testing. For the DSG4 deletion, establish copy number/zygosity and search for a second pathogenic DSG4 allele before causal interpretation.

9. Limitations

  1. The files are synthetic educational data and do not represent a real patient.
  2. No phenotype, age, sex/karyotype, family history, pedigree, segregation, phase, or mosaicism data were provided.
  3. Final SNV VCFs contain six retained records each. Earlier workflow review documented removal of synthetic background records whose REF alleles did not match GRCh38 during bcftools norm -c x.
  4. ClinVar classifications are database assertions and can change; a real clinical report requires date-stamped review and laboratory curation.
  5. A zero SpliceAI score does not show that a missense or nonsense variant is benign; it only indicates no strong predicted splice effect in the reported model output.
  6. ISV is supporting computational evidence. Its value and the selected 0.95 threshold must not override curated clinical evidence.
  7. CNV breakpoints and copy-number state were taken from BED-derived records and were not orthogonally validated.
  8. Population-frequency, phenotype-matching, segregation, functional, and literature evidence were not comprehensively integrated.
  9. The report summarizes pipeline classifications; it does not independently assign a new ACMG/AMP or ACMG/ClinGen classification.
