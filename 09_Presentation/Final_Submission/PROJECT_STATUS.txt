LIMITATIONS
-----------
1. The local Ensembl VEP cache was unavailable during the final rerun.
2. Where current VEP output was unavailable, consequence and HGVS values
   were retained from the validated input VCF and clearly labelled as
   input metadata.
3. Live ClinVar, gnomAD and AlphaMissense evidence was not available for
   every final demonstration variant.
4. AnnotSV, ClassifyCNV, ISV-CNV and ClinGen CNV are represented as an
   integration framework and require external installation, resources or
   manual access before actual execution.
5. OMIM entries are local reference metadata, not a live licensed API query.
6. All outputs are for research and educational use and require review by
   a qualified genetics professional.
